# ART — Guide du Photographe

[![GitHub Pages](https://img.shields.io/badge/docs-GitHub%20Pages-blue)](https://jordanatdown.github.io/ART-Guide-Photographe/)

Documentation complète du logiciel **ART** (Another RawTherapee) orientée photographe. Couvre l'ensemble des modules de traitement RAW dans le contexte d'un workflow **RAW → TIFF → Affinity Photo**.

## Consulter la documentation

**[jordanatdown.github.io/ART-Guide-Photographe](https://jordanatdown.github.io/ART-Guide-Photographe/)**

## Contenu

18 modules documentés :

- Workflow & pipeline de traitement
- Exposition, égaliseur tonal, courbes
- Log Tone Mapping & compression plage dynamique
- Balance des blancs, saturation/vibrance
- Égaliseur de couleurs (HSL), correction des couleurs
- Mélange des canaux, courbes RVB
- Texture, contraste local, lissage
- Looks & presets (Méditerranée)
- Ordre des modules

## Stack

- HTML5 monofichier, sans dépendances
- CSS embarqué (thème sombre, responsive)
- JS minimal (navigation SPA, recherche)
- Déploiement automatique via GitHub Actions → GitHub Pages

## Contribuer

Modifier directement [index.html](index.html) puis pousser sur `main` — le déploiement est automatique.
